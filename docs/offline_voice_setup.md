# Настройка офлайн-голосового ввода

## 1) Добавить модель Vosk
Скачайте русскую модель с сайта Vosk и распакуйте в проект:

`app/src/main/assets/model-ru/`

После распаковки внутри `model-ru` должны быть директории:
- `am/`
- `conf/`
- `graph/`
- `ivector/`

Рекомендуемая стартовая модель: `vosk-model-small-ru-0.22`.

## 2) Проверить разрешение на микрофон
В манифест уже добавлено:

`android.permission.RECORD_AUDIO`

При первом запуске голосового экрана приложение запросит доступ.

## 3) Сценарий работы в приложении
1. Нажать на иконку микрофона.
2. Сказать название поля (например: `фио`, `позывной`, `диагноз`).
3. Продиктовать значение поля.
4. Повторять для следующих полей.
5. Нажать на иконку микрофона повторно — откроется ручная форма с заполненными данными.

## 4) Что уже реализовано
- Оффлайн распознавание на `Vosk`.
- Двухрежимная логика: `команда поля -> значение`.
- Грамматика команд для повышения устойчивости в шуме.
- Автозаполнение тех же полей, что в ручной форме.
